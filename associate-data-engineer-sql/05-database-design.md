# Database Design

## Overview
This module covers database design principles and normalization.

## Topics to Cover
- Database schemas
- Normalization (1NF, 2NF, 3NF)
- Denormalization
- Primary and foreign keys
- Entity-Relationship diagrams
- Data integrity constraints
- Indexing strategies
- Views and stored procedures

- [ ] Understand database normalization forms

- [ ] Optimize database performance with indexes
